🧠 What IS the Neural Network Here?
The Embedding Layer
The neural network has an Embedding Layer as its very first layer. This layer IS the embedding matrix W (26 × 4). It is just a table of learnable numbers — one row per word in the vocabulary.
How Values Are Learned
The values in the matrix start random. During training, the network makes predictions, calculates error, and uses backpropagation + gradient descent to update these values until related words end up with similar vectors.
💡 The Core Analogy
The Embedding Matrix is like a seating chart at a wedding. Initially you assign random seats. But after seeing who needs to sit together (training data), you rearrange seats so related people sit near each other. "learning" and "machine" end up close. "ai" and "intelligence" end up close. The positions themselves encode relationships.

⚠️ Why One-Hot Alone Is Not Enough

Every pair of one-hot vectors has zero dot product — meaning zero similarity. "learning" and "machine" are completely unrelated according to one-hot encoding. This is the problem the embedding matrix solves.

x_learning · x_machine = 0 (no common cells!)
x_learning · x_learning = 1 (same word only)
→ One-hot can't capture any semantic similarity
