What is a Vector?

A vector has both magnitude (size) and direction. A scalar only has magnitude.

From your notes — Speed = 100 km/hr (scalar, no direction). Velocity = 100 km/hr North (vector, has direction)!

Scalars (no direction)

Speed, temperature, mass
Time, distance, energy
Pressure, volume

Vectors (have direction)

Velocity, force, displacement
Acceleration, momentum
Torque, angular velocity

Component Notation

from your whiteboard — OA = 4î + 2ĵOA = 4î + 2ĵ Meaning: 4 units along x-axis (î = [1,0,0]) 2 units along y-axis (ĵ = [0,1,0]) General 3D: v = aî + bĵ + ck̂ where k̂ = [0,0,1] (z-axis direction)

Magnitude Formula

pythagorean theorem — from your notes2D: |v| = sqrt(x² + y²) 3D: |v| = sqrt(x² + y² + z²) Your example: OA = 4î + 2ĵ |OA| = sqrt(4² + 2²) = sqrt(16+4) = sqrt(20) = 2√5 ≈ 4.47

Unit Vector â

A unit vector has magnitude exactly 1. It captures direction only, stripping away size. Written with hat: â ("a hat").

Your whiteboard shows this exactly! â = OA / |OA| = (4î + 2ĵ) / sqrt(20)

Standard Unit Vectors î ĵ k̂

the 3 fundamental directionsî = [1, 0, 0] → x-axis direction |î| = 1 ĵ = [0, 1, 0] → y-axis direction |ĵ| = 1 k̂ = [0, 0, 1] → z-axis direction |k̂| = 1 Your notes: î → unit vector towards X axis ĵ → unit vector towards Y axis

Finding a Unit Vector — Full Steps

your exact whiteboard exampleGiven: OA = 4î + 2ĵ Step 1 — find magnitude: |OA| = sqrt(4² + 2²) = sqrt(20) = 2√5 Step 2 — divide each component by magnitude: â = OA / |OA| = (4î + 2ĵ) / sqrt(20) = (4/sqrt(20))î + (2/sqrt(20))ĵ = 0.8944î + 0.4472ĵ Step 3 — verify (must equal 1): |â| = sqrt(0.8944² + 0.4472²) = sqrt(0.7999 + 0.1999) = sqrt(1.0) = 1 ✓

⚠ Edge Case — Zero Vector

!
Unit vector is undefined for the zero vector.
The zero vector 0⃗ has magnitude = 0. Division by zero is undefined, so â = 0⃗ / 0 has no meaning. Always check |v| ≠ 0 before computing a unit vector.
ruleIf |v| = 0 → â is UNDEFINED (zero vector has no direction) If |v| ≠ 0 → â = v / |v| (always gives |â| = 1)

Triangle Law (Tip-to-Tail)

add component by componenta + b = (a_x+b_x)î + (a_y+b_y)ĵ Example: a = 3î + 2ĵ, b = 1î + 4ĵ a + b = (3+1)î + (2+4)ĵ = 4î + 6ĵ |a+b| = sqrt(16+36) = sqrt(52) ≈ 7.21

Parallelogram Law

magnitude when angle θ between a and b|R|² = |a|² + |b|² + 2|a||b|cos(θ) Special cases: θ = 0°: |R| = |a| + |b| (maximum) θ = 90°: |R| = sqrt(|a|² + |b|²) (Pythagoras) θ = 180°: |R| = | |a| − |b| | (minimum)

Scalar Multiplication

multiply every component by kk × (aî+bĵ) = (ka)î + (kb)ĵ k > 0 → same direction, different size k < 0 → REVERSES direction k = 0 → zero vector v = 2î + 3ĵ 3v = 6î + 9ĵ (tripled) −v = −2î − 3ĵ (reversed) 0.5v = 1î + 1.5ĵ (halved)

Angle Between Two Vectors

formula — from dot product definitioncos(θ) = (a · b) / (|a| × |b|) θ = arccos( (a · b) / (|a| × |b|) ) Range: 0° ≤ θ ≤ 180°

Full Worked Example

a = [1,2,2] and b = [2,1,−2]Step 1 — dot product: a·b = 1×2 + 2×1 + 2×(−2) = 2 + 2 − 4 = 0 Step 2 — magnitudes: |a| = sqrt(1+4+4) = sqrt(9) = 3 |b| = sqrt(4+1+4) = sqrt(9) = 3 Step 3: cos(θ) = 0 / (3 × 3) = 0 θ = arccos(0) = 90° ← PERPENDICULAR!

Direction Cosines

angles with each axisv = aî+bĵ+ck̂, |v| = r cos(α) = a/r ← angle with x-axis cos(β) = b/r ← angle with y-axis cos(γ) = c/r ← angle with z-axis Identity (ALWAYS true): cos²α + cos²β + cos²γ = 1 Example: v=[1,2,2], r=3 1/9 + 4/9 + 4/9 = 9/9 = 1 ✓

Dot Product (Scalar Product)

Result is a single number. Measures alignment. Used in AI attention, work calculations, and similarity.

algebraic formulaa·b = a₁b₁ + a₂b₂ + a₃b₃
geometric formulaa·b = |a| |b| cos(θ)

What the Result Tells You

a · b > 0

Angle < 90° — pointing same way

a · b = 0

Angle = 90° — perpendicular!

a · b < 0

Angle > 90° — opposing direction

a · a = |a|²

Dot with itself = magnitude²

Cross Product (Vector Product)

Result is a vector perpendicular to both inputs. Only in 3D. Used for area, torque, and surface normals.

magnitude = area of parallelogram|a×b| = |a| |b| sin(θ) ← perpendicular when θ=90° Direction: right-hand rule (curl fingers a→b, thumb = direction of a×b)
determinant formulaa×b = | î ĵ k̂ | | a₁ a₂ a₃ | | b₁ b₂ b₃ | = î(a₂b₃−a₃b₂) − ĵ(a₁b₃−a₃b₁) + k̂(a₁b₂−a₂b₁)

Worked Example

a = 2î+3ĵ+k̂, b = î+2ĵ+4k̂î: 3×4 − 1×2 = 10 ĵ: −(2×4 − 1×1) = −7 k̂: 2×2 − 3×1 = 1 a×b = 10î − 7ĵ + k̂ |a×b| = sqrt(150) = 5√6 (area of parallelogram) Check: (a×b)·a = 20−21+1 = 0 ✓ (perpendicular)

Triple Products

Scalar Triple Product [a b c]

definition[a b c] = a · (b × c) = | a₁ a₂ a₃ | | b₁ b₂ b₃ | | c₁ c₂ c₃ | |[a b c]| = VOLUME of parallelepiped with edges a, b, c [a b c] = 0 → vectors are COPLANAR ⚠ Valid only when all three vectors share the same origin

Worked Example

a=î−ĵ+k̂, b=2î+3ĵ−k̂, c=−î−ĵ+5k̂[a b c] = | 1 −1 1 | | 2 3 −1 | | −1 −1 5 | = 1(15−1) − (−1)(10−1) + 1(−2+3) = 14 + 9 + 1 = 24 Volume = |24| = 24 cubic units

Properties

1
Cyclic unchanged: [a b c] = [b c a] = [c a b]
2
Anti-cyclic changes sign: [a b c] = −[a c b]
3
Coplanar test: [a b c] = 0 ⟺ coplanar (requires all three vectors from same origin)
4
Interchangeable: a·(b×c) = (a×b)·c

Vector Triple Product — BAC-CAB Rule

a × (b × c) = b(a·c) − c(a·b)Result: VECTOR lying in the plane of b and c. NOT associative: a×(b×c) ≠ (a×b)×c

Vector Projection

How much of one vector lies along the direction of another.

Scalar Projection — A Number

how far a "reaches" in direction of bscalar_proj = (a·b) / |b| = |a| cos(θ) Example: a=[3,4], b=[1,0] scalar_proj = (3×1+4×0) / 1 = 3

Vector Projection — A Vector

component of a along bvector_proj = (a·b / b·b) × b Example: a=[3,4], b=[2,1] a·b=10, b·b=5 proj = (10/5)×[2,1] = [4,2] Perpendicular component: perp = a − proj = [3,4]−[4,2] = [−1,2] Check: proj·perp = −4+4 = 0 ✓

Section Formula

Find position vector of point P that divides line segment AB in ratio m:n.

Internal Division

P divides AB internally in ratio m:nOP = (m × OB + n × OA) / (m + n) Example: A=(1,2), B=(7,5), ratio 2:1 OP = (2×[7,5] + 1×[1,2]) / 3 = [5, 4]

External Division

P divides AB externally in ratio m:nOP = (m × OB − n × OA) / (m − n) Example: A=(1,2), B=(7,5), ratio 3:1 externally OP = (3×[7,5] − 1×[1,2]) / 2 = [10, 6.5]

Midpoint (m = n = 1)

midpoint formulaM = (OA + OB) / 2 Example: A=(2,4), B=(6,8) M = [4, 6] ✓

Transformer Attention = Dot Product

core formula of GPT, BERT, ClaudeAttention(Q,K,V) = softmax( Q·Kᵀ / sqrt(d_k) ) × V Q = Query "What am I looking for?" K = Key "What does each token offer?" V = Value "What info to pass forward?" Q·Kᵀ = matrix of DOT PRODUCTS = attention scores! sqrt(d_k) = variance stabilization, NOT normalization. When d_k is large, dot products Q·Kᵀ grow in magnitude, pushing softmax into regions with near-zero gradients (very flat distribution → vanishing updates during training). Dividing by sqrt(d_k) keeps variance ≈ 1 regardless of d_k.
